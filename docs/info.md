## How it works

32 switches to represent BCD numbers. The Circuit lights up a message of birthday wishes when the input number has a prefix of 2904 and is followed by a number grater or eual than 1995.

## How to test
Switches configuration to light the message up: 0010 1001 0000 0100 (0001 1001 1001 0101 or grater). You can also try to input an invalid number and observe it doesnt work.

## External hardware
I used 9 seven segments displays (cathode)
